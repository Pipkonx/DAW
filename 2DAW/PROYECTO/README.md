## 💎 [FintechPro — Gestión Patrimonial Inteligente](./anteproyecto/)

FintechPro es una aplicación de grado empresarial diseñada para la gestión consolidada de activos financieros (acciones, ETFs, cripto, fondos y efectivo).

### Características Principales:
- **Tecnologías**: Laravel 12 + Vue 3 + Inertia.js (VILT Stack).
- **Seguridad**: Autenticación 2FA, auditoría de sesiones y geolocalización.
- **Automatización**: Ingesta de datos mediante OCR y procesamiento de PDFs bancarios.
- **Mercados**: Telemetría en tiempo real con motor de scraping resiliente.
