Detailed Description

Página personalizada para que los usuarios editen su perfil.

Permite a los usuarios actualizar su información básica, cambiar su contraseña y subir una foto de perfil (avatar). También incluye campos específicos según el rol del usuario (Alumno, Tutor, etc.).
