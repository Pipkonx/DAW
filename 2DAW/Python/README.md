Este módulo sirve como introducción a la programación con Python, explorando su versatilidad para el scripting, el análisis de datos y el desarrollo de herramientas de backend.

## 📁 Temas y Actividades

### 📘 Bloques de Aprendizaje
- **[T1 - Fundamentos](./T1/)**: Introducción a la sintaxis básica. Variables, tipos de datos dinámicos, operadores lógicos y de comparación.
- **[T2 - Control de Flujo](./T2/)**: Estructuras condicionales y bucles. Gestión de listas, tuplas y diccionarios para el procesamiento de datos.
- **[T3 - Programación Modular](./T3/)**: Creación de funciones, gestión de módulos externos y manejo de excepciones.

### 🏆 Evaluación
- **[Actividad Evaluable](./Actividad%20Evaluable/)**: Proyecto integrador de Python donde se aplica la lógica de programación para resolver un problema de automatización o gestión de datos.

### 🌟 Actividades Complementarias
- **[Actividades Faltantes](./Actividades_Faltantes/)**: Colección de 9 actividades avanzadas que incluyen:
  - 📊 **Pandas y NumPy**: Análisis y simulación de datos.
  - 🖥️ **Tkinter**: Aplicaciones de escritorio (IMC, CRUD).
  - 🌐 **Flask**: Creación de APIs REST.
  - 📈 **Matplotlib**: Visualización de datos.
  - 🤖 **Scikit-learn**: Inteligencia Artificial básica.
  - 🎮 **PyGame**: Desarrollo de videojuegos.
