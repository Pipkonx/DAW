## 🔧 Configuración del Entorno

1. **Instalar Python**:
   - Descarga e instala Python desde [python.org](https://www.python.org/downloads/)
   - Asegúrate de marcar la opción "Add Python to PATH" durante la instalación

2. **Verificar la instalación**:
   ```bash
   python --version
   ```
