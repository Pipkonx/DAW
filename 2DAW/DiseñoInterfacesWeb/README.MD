Este módulo se enfoca en la creación de interfaces de usuario atractivas, funcionales y accesibles, utilizando estándares modernos de diseño y estilos CSS avanzados.

## 📁 Detalle de Actividades y Proyectos

### 🏗️ Fundamentos de Diseño
- **[01_Introduccion](./01_Introduccion/)**: Conceptos básicos de diseño web, tipografía y colorimetría.
- **[02_Caja](./02_Caja/)**: El modelo de caja de CSS (Box Model). Márgenes, rellenos, bordes y el cálculo de dimensiones.
- **[03_Selectores](./03_Selectores/)**: Dominio de selectores avanzados, pseudo-clases y pseudo-elementos para un estilado preciso.

### 📐 Layouts y Estructuras
- **[04_Flexbox](./04_Flexbox/)**: Diseño unidimensional. Distribución de elementos en filas o columnas, alineación y orden dinámico.
- **[05_Grid](./05_Grid/)**: Diseño bidimensional. Creación de rejillas complejas, áreas de grid y diseño responsive avanzado.
- **[11_Maquetacion](./11_Maquetacion/)**: Proyecto integrador de maquetación web completa siguiendo un diseño previo.

### 🎨 Estilos Avanzados y Herramientas
- **[06_Sass](./06_Sass/)**: Uso del preprocesador SASS. Mixins, variables, anidamiento y herencia para CSS escalable.
- **[12_Figma](./12_Figma/)**: Diseño y prototipado. Creación de wireframes y maquetas de alta fidelidad antes de codificar.
- **[16_Boostrap](./16_Boostrap/)**: Implementación de frameworks de terceros. Uso de componentes, sistema de rejilla y utilidades de Bootstrap 5.

### ✨ Visualización e Interactividad
- **[08_Animaciones](./08_Animaciones/)**: Transiciones y Keyframes. Creación de micro-interacciones y animaciones que mejoran la experiencia de usuario.
- **[09_SVG](./09_SVG/)**: Uso de gráficos vectoriales. Optimización, manipulación por CSS y animación de iconos.
- **[10_Canvas](./10_Canvas/)**: Dibujo y animaciones programáticas mediante la API de Canvas de HTML5.

### 🚀 Proyectos Finales
- **[13_Portfolio](./13_Portfolio/)**: Diseño y desarrollo de marca personal y portfolio de proyectos.
- **[15_1_PROYECTO](./15_1_PROYECTO/)**: Proyecto integrador de diseño donde se aplican todas las técnicas del curso.
