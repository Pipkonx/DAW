**The Core Hunter** es un videojuego de acción y plataformas desarrollado íntegramente en el navegador, diseñado para poner a prueba la lógica de colisiones, gestión de estados y renderizado gráfico en tiempo real.

## 🚀 Características Técnicas
- **Motor**: HTML5 Canvas API (renderizado 2D).
- **Lenguaje**: JavaScript Nativo (ES6+).
- **Mecánicas**: 
    - Movimiento lateral y salto parabólico.
    - Sistema de detección de colisiones por cajas (AABB).
    - Generación de enemigos y recolección de objetos ("The Cores").
    - Sistema de puntuación y niveles de dificultad progresiva.

## 🎨 Aspecto Visual
El juego utiliza un estilo Pixel Art retro, con sprites animados y fondos en parallax para dar sensación de profundidad.
