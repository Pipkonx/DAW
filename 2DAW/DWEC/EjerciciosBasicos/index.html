Ejercicios de

JS

Ejercicio 1

Script que tome una palabra y una letra mediante dos prompts consecutivos. A continuación deberá decirnos cuántas veces aparece la letra solicitda en esa palabra

Ejercicio 2

Solicitamos una frase formada por letras y números. Y decirnos cuántas letras contine, cuantos números y la suma de éstos

Ejercicio 3

Script que genere un número aleatorio y nos vaya preguntando que número es. En cada paso el programa nos indicará si el número a acertar es menor, mayor o igual que el que hemos tecleado. El script finalizará cuando acertemos el número.

Ejercicio 4

Script que nos diga si una palabra es palíndroma (se escribe igual de izquierda a derecha)

Ejercicio 5

Crear una funcion que indique una operación sumar, restar, multiplicar, dividir, aplica a todos los numeros y devuelvelo.

Ejercicio 6

Tenemos un array y agregamos un nuevo elemento al final del Array y otro botón que nos permita eliminar el último elemento de array
