Ejercicios de JS

  1. Creamos un Script que tome una palabra y una letra mediante dos prompts consecutivos. A continuación deberá decirnos cuántas veces aparece la letra solicitda en esa palabra
  2. Crear un script quee nos solicite una frase formada por letras y números. Finalmente debe decirnos cuántas letras contine, cuantos números y la suma de éstos
  3. Crear un script que genere un número aleatorio y nos vaya preguntando que número es. En cada paso el programa nos indicará si el número a acertar es menor, mayor o igual que el que hemos tecleado. El script finalizará cuando acertemos el número.
  4. Crear un script que nos diga si una palabra ees palíndroma (se escribe igual de izquierda a derecha)
