# Actividad 3: Interfaces Dinámicas (Vue.js y JavaScript)

Este bloque se centra en mejorar la experiencia de usuario (UX) mediante interacciones sin recarga de página, utilizando diferentes niveles de integración de JavaScript y Vue.

### 🟢 Problema 3: Interactividad y Frontend Moderno
Se han implementado tres soluciones distintas para cubrir los requisitos del curso:

*   **3.1 JavaScript Nativo y DataTables**: Gestión de clientes con Fetch API y renderizado dinámico en [index_js.blade.php](resources/views/clients/index_js.blade.php).
*   **3.2 Vue 3 + Quasar (CDN)**: Interfaz enriquecida con componentes premium de Quasar en [index_quasar.blade.php](resources/views/clients/index_quasar.blade.php).
*   **3.3 Modern Web Stack (Vite + Inertia)**: Aplicación SPA moderna usando componentes de Vue 3 aislados y compilados con Vite. Ver [Index.vue](resources/js/Pages/Tasks/Index.vue).
